DETAILING WEBSITE

How to use:
1. Unzip this folder.
2. Open index.html in a browser to preview the website.
3. Edit index.html to change:
   - Business name
   - Phone number
   - Email
   - Location
   - Package names/prices
4. Upload the folder to a website host like Netlify, Vercel, GitHub Pages, or your own domain host.

Main files:
- index.html = website content
- styles.css = website design
